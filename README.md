# Spring Boot Job Search Application Using Supabase

Verify the project:
```bash
./mvnw clean verify
```
Build (optional but explicit) and run the app in JIT mode:

```bash
./mvnw clean package
```
```bash
./mvnw spring-boot:run
```
OR run the JAR:
```bash
java -jar target/supabase-spring-job-search-0.0.1.jar
```

Build and run the app in native mode:

```bash
./mvnw -Pnative native:compile
```
```bash
./target/supabase-spring-job-search
```

Open [_http://localhost:8080](http://localhost:8080) to access the application.
